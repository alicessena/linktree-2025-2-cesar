# Central de Links - Turma ADS

> Plataforma web desenvolvida para organizar horários, contatos e calendário acadêmico da turma de Análise e Desenvolvimento de Sistemas.

---

## Sobre o Projeto

A **Central de Links - Turma ADS** é uma aplicação web responsiva criada para facilitar o acesso às principais informações acadêmicas da turma, como:

- Horário de Aulas
- Calendário Acadêmico Interativo
- Links úteis
- Contato direto com docentes
- Acesso às cadeiras

O projeto foi desenvolvido com foco em **organização, usabilidade e experiência visual moderna**.

---

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis e design responsivo)
- JavaScript
- Grid Layout
- Design Dark Mode com Glassmorphism

---

## Funcionalidades

✔ Calendário acadêmico mensal  
✔ Dias com eventos destacados  
✔ Navegação com scroll suave  
✔ Lista completa de eventos  
✔ Botões de e-mail personalizados  
✔ Layout responsivo  
✔ Interface moderna com animações suaves

Este projeto foi desenvolvido para:

- Melhorar a organização acadêmica
